Subject: Unix System Programming

Subject Code: BCS515C

Command Reference

1. Change Password (passwd)
Command: passwd
Description: This command is used to change the password of the currently logged-in user. When executed, it prompts for the current password, then asks for the new password and confirmation.
Steps:
  1. Type passwd and press Enter.
  2. Enter the current password when prompted.
  3. Enter the new password and confirm it.
2. Find Username (whoami)
Command: whoami
Description: Displays the username of the currently logged-in user.
Usage: Just type whoami in the terminal and press Enter.
3. Get User ID (id)
Command: id
Description: Another way to get the username of the current user using the id command.
